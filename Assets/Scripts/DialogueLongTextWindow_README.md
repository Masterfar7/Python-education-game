# Long Text Window - Инструкция по настройке

## Описание
Отдельное окно для длинных текстов, которое появляется вместо обычного диалогового облачка когда стоит галочка `Enable Scroll`.

## Как работает
- **Галочка Enable Scroll выключена** → обычное диалоговое облачко
- **Галочка Enable Scroll включена** → отдельное окно для длинного текста

## Настройка в Unity

### 1. Создание UI для длинного текста

1. **Создайте Panel:**
   - Правый клик в Canvas → UI → Panel
   - Назовите "LongTextPanel"
   - Сделайте его на весь экран или большую часть

2. **Настройте Panel:**
   - Цвет фона: полупрозрачный черный (0, 0, 0, 200)
   - Anchors: Stretch-Stretch (растянуть на весь экран)

3. **Создайте внутренний контейнер:**
   - Правый клик на LongTextPanel → UI → Panel
   - Назовите "ContentPanel"
   - Настройте размер и позицию (например, 80% экрана по центру)

4. **Добавьте текстовое поле:**
   - Правый клик на ContentPanel → UI → Text - TextMeshPro
   - Назовите "LongTextContent"
   - Настройте:
     - Font Size: 24-32
     - Alignment: Top-Left
     - Wrapping: Enabled
     - Overflow: Overflow

5. **Добавьте кнопку закрытия:**
   - Правый клик на ContentPanel → UI → Button - TextMeshPro
   - Назовите "CloseButton"
   - Текст: "Далее" или "→"
   - Разместите внизу справа

### 2. Добавление скрипта

1. **Добавьте компонент:**
   - Выберите LongTextPanel
   - Add Component → Dialogue Long Text Window

2. **Настройте ссылки:**
   - Long Text Panel: перетащите LongTextPanel
   - Long Text Content: перетащите LongTextContent
   - Close Button: перетащите CloseButton
   - Typing Speed: 0.03 (или другая скорость)

3. **Скройте панель по умолчанию:**
   - Выберите LongTextPanel
   - Снимите галочку слева от имени (деактивируйте)

### 3. Подключение к DialogueManager

1. Откройте объект с DialogueManager
2. В секции **Long Text Window:**
   - Long Text Window: перетащите LongTextPanel

### 4. Использование

1. Откройте NPC с диалогом
2. Найдите реплику с длинным текстом
3. Поставьте галочку **Enable Scroll** ✓
4. Готово!

## Поведение

### Обычная реплика (Enable Scroll = ✗)
```
[Диалоговое облачко]
Кай: Привет!
[Кнопка "Далее"]
```

### Длинная реплика (Enable Scroll = ✓)
```
[Полноэкранное окно]
Добро пожаловать в Сад Вечного Цветения.
Здесь царят циклы. Без них растения не растут,
кристаллы не горят, а сорняки заполонят всё...
[Кнопка "Далее"]
```

## Логика кнопки "Далее"

1. **Первый клик** (текст печатается):
   - Показывает весь текст мгновенно

2. **Второй клик** (текст напечатан):
   - Закрывает окно
   - Переходит к следующей реплике

## Стилизация

### Пример красивого окна:

**LongTextPanel:**
- Background: темный с прозрачностью
- Можно добавить Image для рамки

**ContentPanel:**
- Background: светлый (пергамент, бумага)
- Добавьте Image со спрайтом рамки
- Padding: 20-30 пикселей со всех сторон

**LongTextContent:**
- Font: красивый читаемый шрифт
- Color: темный (для контраста с фоном)
- Line Spacing: 1.1-1.2 (для читаемости)

**CloseButton:**
- Стилизуйте под общий дизайн игры
- Можно добавить иконку стрелки

## Преимущества

✅ Простота - одна галочка на реплику
✅ Чистота - обычные реплики не затронуты
✅ Гибкость - можно стилизовать окно как угодно
✅ Читаемость - длинный текст в большом окне
✅ Без скролла - весь текст виден сразу (или добавьте ScrollRect если нужно)

## Добавление скролла в Long Text Window (опционально)

Если текст ОЧЕНЬ длинный и не помещается даже в большое окно:

1. Оберните LongTextContent в ScrollView
2. Настройте ScrollRect на ContentPanel
3. Текст будет прокручиваться внутри окна

## Troubleshooting

**Проблема:** Окно не появляется
- Проверьте, что Long Text Window подключен в DialogueManager
- Убедитесь, что галочка Enable Scroll включена на реплике

**Проблема:** Текст не виден
- Проверьте цвет текста (должен контрастировать с фоном)
- Убедитесь, что LongTextContent внутри ContentPanel

**Проблема:** Кнопка не работает
- Проверьте, что Close Button подключен в инспекторе
- Убедитесь, что на кнопке есть компонент Button

**Проблема:** Облачко и окно показываются одновременно
- Скрипт автоматически скрывает облачко, проверьте логи на ошибки
