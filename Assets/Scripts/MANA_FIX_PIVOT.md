# Исправление: Мана уменьшается с двух сторон

## Проблема:
Заполнение маны уменьшается с обеих сторон (центр), а должно только справа.

## Решение в Unity:

### 1. Выберите объект ManaFill в Hierarchy

### 2. В Inspector → RectTransform настройте:

**Anchors (якоря):**
- Min: X = 0, Y = 0.5
- Max: X = 0, Y = 0.5
- (Или выберите preset: Left-Center)

**Pivot (точка опоры):**
- X = **0** (ВАЖНО!)
- Y = 0.5

**Position:**
- Pos X = 0
- Pos Y = 0

### 3. Визуально должно быть так:

```
Inspector → RectTransform:
┌─────────────────────────────┐
│ Anchors:                    │
│   Min: (0, 0.5)             │
│   Max: (0, 0.5)             │
│                             │
│ Pivot:                      │
│   X: 0  ◄── НОЛЬ!           │
│   Y: 0.5                    │
│                             │
│ Position:                   │
│   X: 0                      │
│   Y: 0                      │
│                             │
│ Width: 200                  │
│ Height: 30                  │
└─────────────────────────────┘
```

## Как это работает:

- **Pivot X = 0** → точка опоры слева
- При изменении Width заполнение уменьшается **только справа**
- Левый край остаётся на месте

## Проверка:
После настройки запустите игру - мана должна уменьшаться только справа! ◄━━━━━
