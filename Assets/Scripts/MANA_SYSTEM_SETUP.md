# Система испытаний с маной - Инструкция по настройке (обновлено)

## Как работает система

### Механика:
1. При активации испытания с маной появляется **шкала маны** (слева вверху)
2. Сорняк начинает **расти** (анимация 2 сек)
3. С каждой фразой диалога **мана уменьшается** (плавная анимация)
4. Цвет заполнения меняется: голубой → жёлтый → красный
5. Когда игрок вводит правильный код:
   - Сорняк **гибнет** (коричневеет и уменьшается)
   - Пустая шкала маны остаётся видимой
6. При завершении диалога всё исчезает

## Настройка в Unity

### Шаг 1: Создание UI шкалы маны

1. **Создайте Canvas** (если нет)
   - Hierarchy → ПКМ → UI → Canvas

2. **Создайте ManaUI** (корневой объект)
   - ПКМ на Canvas → Create Empty
   - Назовите "ManaUI"
   - RectTransform → Anchor: **Top-Left** (левый верхний угол)
   - Position: X=150, Y=-80

3. **Добавьте компоненты на ManaUI:**
   - Add Component → **ManaUI**
   - Add Component → **Canvas Group**

4. **Создайте фон шкалы (внешняя часть):**
   - ПКМ на ManaUI → UI → Image
   - Назовите "ManaBackground"
   - Sprite: рамка/контур шкалы (тёмный цвет)
   - Размер: Width=200, Height=30
   - Image Type: **Sliced** (если используете 9-slice спрайт)

5. **Создайте заполнение шкалы:**
   - ПКМ на ManaUI → UI → Image
   - Назовите "ManaFill"
   - Sprite: заполнение (яркий цвет)
   - Размер: такой же как Background (200x30)
   - Image Type: **Filled**
   - Fill Method: **Horizontal**
   - Fill Origin: **Left**
   - Fill Amount: **1** (полная)

6. **Настройте компонент ManaUI:**
   - Mana Fill Image → перетащите **ManaFill**
   - Mana Background Image → перетащите **ManaBackground**
   - Canvas Group → перетащите **CanvasGroup**
   - Max Mana → 100
   - Mana Decrease Per Line → 10 (по умолчанию)
   - Full Mana Color → голубой (Cyan)
   - Half Mana Color → жёлтый (Yellow)
   - Low Mana Color → красный (Red)

### Шаг 2: Создание сорняка

1. **Создайте GameObject "Weed"** на сцене
2. Добавьте **SpriteRenderer** с спрайтом сорняка
3. Добавьте компонент **WeedAnimationController**
4. Настройте параметры:
   - Grow Duration → 2 сек
   - Target Scale → (1, 1, 1) или больше
   - Grow Curve → оставьте по умолчанию или настройте
   - Death Duration → 1.5 сек
   - Death Color → коричневый (0.5, 0.3, 0.1)
   - Sprite Renderer → назначьте SpriteRenderer

### Шаг 3: Настройка DialogueManager

1. Найдите GameObject с **DialogueManager**
2. В секции "Mana Challenge System":
   - Mana UI → перетащите объект **ManaUI**
   - Weed Animation → перетащите объект **Weed**

### Шаг 4: Настройка диалога с испытанием

1. Откройте NPC с диалогом
2. Найдите **первую реплику** испытания:
   - ✅ **Is Mana Challenge** - включить испытание с маной
   - **Mana Decrease Amount** → 0 (на первой фразе не отнимаем)

3. На **последующих фразах** (пока игрок не выполнит задание):
   - **Mana Decrease Amount** → 10 (или другое значение)
   - Это количество маны будет отниматься после печати каждой фразы

4. На **реплике с заданием**:
   - ✅ **Start Task After Line** - запустить задание
   - **Mana Decrease Amount** → 0 (не отнимаем на фразе с заданием)

## Пример настройки диалога

```
Реплика 1: "Приготовься к испытанию!"
  ✅ Is Mana Challenge = true
  Mana Decrease Amount = 0

Реплика 2: "У тебя есть ограниченное время..."
  Mana Decrease Amount = 15

Реплика 3: "Напиши код, чтобы победить сорняк!"
  ✅ Start Task After Line = true
  Mana Decrease Amount = 0
```

## Логика работы

- **Мана уменьшается** после печати каждой фразы (если `manaDecreaseAmount > 0`)
- **Цвет заполнения** меняется автоматически:
  - 100%-50% → голубой → жёлтый
  - 50%-0% → жёлтый → красный
- **При выполнении задания:**
  - Сорняк гибнет (независимо от количества маны)
  - Пустая шкала остаётся видимой
- **При завершении диалога:**
  - UI маны исчезает
  - Сорняк сбрасывается

## Дополнительные возможности

### Проверка маны из кода:
```csharp
ManaUI manaUI = FindObjectOfType<ManaUI>();

// Проверить, закончилась ли мана
if (manaUI.IsManaEmpty())
{
    Debug.Log("Мана закончилась!");
}

// Получить процент маны (0-1)
float percent = manaUI.GetManaPercent();
```

### Настройка количества маны за фразу:
```csharp
// Установить глобальное значение
manaUI.SetManaDecreasePerLine(20);

// Или настроить индивидуально для каждой фразы в Inspector
```

## Требования

- **TextMeshPro** - НЕ требуется (убран текст)
- **Image с Fill Type** - для заполнения шкалы
- Встроенные корутины Unity (без DOTween)

## Визуальные эффекты

- Плавное уменьшение заполнения (0.3 сек)
- Плавное изменение цвета
- Анимация роста сорняка (2 сек, кривая)
- Анимация гибели сорняка (1.5 сек, коричневый цвет)
