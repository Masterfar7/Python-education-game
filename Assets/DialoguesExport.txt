Над Каем появляются три призрачных силуэта духов.

Ты должен призвать духов по имени. Для этого создай функцию greet(spirit), которая вернёт строку-призыв. 

Затем вызови её для трёх духов.

Скелет:

def greet(spirit):
    # верни строку вида "Дух X, приди ко мне!"
    return f"Дух {spirit}, приди ко мне!"

# Вызови для "Огня", "Ветра", "Воды" и напечатай результат


Ух ты! Одна функция — три призыва. А если бы их было сто?

Ты бы просто вызывал в цикле. Но это уже другая история. Идём дальше.

Теперь нужно активировать руны. Напиши функцию activate_rune(x, y, power), которая ставит защиту. 

Параметры: x (координата), y (координата), power (сила от 1 до 10). Функция пусть печатает: «Руна на (x, y) активирована с силой power».

Вызови её три раза с разными аргументами. Например: (2,3,5), (7,1,8), (4,6,3).

def activate_rune(x, y, power):
    print(f"Руна на ({x},{y}) активирована с силой {power}")

Просто передал разные числа — и каждая руна сработала по-своему. Удобно!

Параметры делают функцию гибкой. Теперь самое сложное.

 в центре зала появляется магический портал, требующим код.

Чтобы открыть финальный портал, нужно создать функцию open_portal(code). Она должна:
1. Проверить длину code (если меньше 5 символов — вернуть "Код слишком короткий").
2. Если длина подходит, зашифровать код: перебрать буквы в цикле и каждую заменить на следующую по алфавиту (a→b, b→c, … z→a).
3. Вернуть зашифрованную строку.
Портал откроется, если результат будет равен "gpsju" (это шифр от "froit").
Вот скелет.

def open_portal(code):
    if len(code) < 5:
        return "Код слишком короткий"
    encrypted = ""
    for ch in code:
        # сдвиг на 1: ord(ch) -> новый символ
        # подсказка: if ch == 'z': new_ch = 'a' else: new_ch = chr(ord(ch)+1)
        encrypted += new_ch
    return encrypted

# Вызов
result = open_portal("froit")
print("Результат шифрования:", result)


Сработало! Я внутри функции использовал if и for — и она сама всё сделала.


Ты понял главное: функция — это маленькая программа внутри программы. 

Она получает данные (параметры), обрабатывает их (тело), и возвращает результат. Теперь портал открыт.

(внутренний монолог) def... это как создать своё заклинание. Называешь его, даёшь ингредиенты (параметры), и оно колдует.
