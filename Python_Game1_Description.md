# Python Game 1 - Полный пайплайн разработки

## Техническое задание

Игра представляет собой образовательную 2D-игру на Unity, где изучение Python становится частью увлекательного путешествия. Игрок в роли Кая исследует мир вместе с гидом Аргусом, решая задачи программирования через встроенный интерпретатор Python прямо в игре. Игра разделена на главы, каждая из которых последовательно вводит новые концепции программирования — от простых команд print() и переменных до условий, циклов, функций и f-строк.

В данном разделе приводятся требования к реализуемой программе. Python Game 1 представляет собой образовательную 2D RPG с акцентом на интерактивное обучение программированию через практические задания. Игрок управляет персонажем, перемещается по уровням, взаимодействует с NPC и решает задачи по программированию для продвижения по сюжету.

Игровой процесс реализован как сочетание исследования мира и решения задач, где игрок должен проходить главы, выполняя задания: на первом уровне необходимо освоить базовые команды print() и переменные, на втором — изучить условия и циклы с бонусом за скорость, на третьем — освоить функции и f-строки для финала. Управление осуществляется через стандартный ввод (клавиатура), обеспечивая плавное перемещение персонажа. Встроенный интерпретатор Python, доступный в редакторе кода, позволяет выполнять реальный Python-код для решения игровых задач.

Персонаж игрока обладает анимациями, отражающими движение, с возможностью взаимодействия с объектами мира. NPC представляют собой гида Аргуса и других персонажей, которые дают задания и помогают в обучении. Диалоговая система поддерживает кинематографические эффекты: персонажи могут идти и разговаривать одновременно, телепортироваться между локациями, а после выполнения заданий появляются духи с анимацией.

Игрок может взаимодействовать с объектами для прохождения уровней, например, кодами и триггерами заданий, с восстанавливающимися предметами. Система заданий отслеживает прогресс игрока, начисляя баллы за правильные решения. Редактор кода включает подсветку синтаксиса, многострочный ввод, вывод результатов и очистку консоли. Система сохранений автоматически сохраняет прогресс между сценами. Магазин (если реализован) открывается при взаимодействии, предоставляя интерфейс для приобретения улучшений. Интерфейс включает элементы управления, редактор кода, меню паузы и статистику. Проект состоит из главного меню и уровней с переходами, оптимизированными для производительности. Особенности включают встроенный интерпретатор Python, интегрированную систему обучения, динамические задания и разнообразие глав.
