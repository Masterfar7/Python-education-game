# Python Learning Game

Образовательная 2D игра на Unity для обучения программированию на Python через интерактивные задания и диалоги с NPC.

## 📋 Описание

Игра представляет собой 2D RPG, где игрок изучает основы программирования на Python, решая задачи и взаимодействуя с персонажами. Встроенный интерпретатор Python позволяет выполнять код прямо в игре.

## ✨ Основные возможности

### Обучение программированию
- **Встроенный Python интерпретатор** - выполнение кода в реальном времени
- **Интерактивные задания** - пошаговое обучение основам Python
- **Система подсказок** - помощь при решении задач
- **Проверка решений** - автоматическая валидация кода

### Игровые механики
- **Диалоговая система** - общение с NPC
- **Система прогресса** - отслеживание выполненных заданий
- **Сохранения** - автоматическое сохранение прогресса
- **Множество уровней** - Chapter 1, Chapter 2 и другие

### Редактор кода
- Подсветка синтаксиса
- Вывод результатов выполнения
- Очистка консоли
- Многострочный ввод кода

## 🎮 Игровой процесс

1. **Исследование мира** - перемещение по 2D локациям
2. **Диалоги с NPC** - получение заданий по программированию
3. **Решение задач** - написание Python кода в встроенном редакторе
4. **Проверка решения** - автоматическая проверка правильности
5. **Прогресс** - переход к следующим заданиям
