# Python Learning Game

Образовательная 2D игра на Unity для обучения программированию на Python через интерактивные задания и диалоги с NPC.

## 📋 Описание

Игра представляет собой 2D RPG, где игрок изучает основы программирования на Python, решая задачи и взаимодействуя с персонажами. Встроенный интерпретатор Python позволяет выполнять код прямо в игре.

## ✨ Основные возможности

### Обучение программированию
- **Встроенный Python интерпретатор** - выполнение кода в реальном времени
- **Интерактивные задания** - пошаговое обучение основам Python
- **Система подсказок** - помощь при решении задач
- **Проверка решений** - автоматическая валидация кода

### Игровые механики
- **Диалоговая система** - общение с NPC
- **Система прогресса** - отслеживание выполненных заданий
- **Сохранения** - автоматическое сохранение прогресса
- **Множество уровней** - Chapter 1, Chapter 2 и другие

### Редактор кода
- Подсветка синтаксиса
- Вывод результатов выполнения
- Очистка консоли
- Многострочный ввод кода

## 🎮 Игровой процесс

1. **Исследование мира** - перемещение по 2D локациям
2. **Диалоги с NPC** - получение заданий по программированию
3. **Решение задач** - написание Python кода в встроенном редакторе
4. **Проверка решения** - автоматическая проверка правильности
5. **Прогресс** - переход к следующим заданиям

## 🔧 Технические детали

### Используемые технологии
- **Unity**: 2021.x+
- **C#**: .NET Standard 2.1
- **TextMeshPro**: UI текст
- **Tilemap**: 2D уровни
- **Universal RP**: Рендеринг

### Ключевые компоненты

#### InterpreterEngine
Встроенный интерпретатор Python, поддерживающий:
- Переменные и типы данных
- Условные операторы (if/elif/else)
- Циклы (for, while)
- Функции print()
- Списки и операции с ними
- Математические операции

#### CodeEditor
UI редактор кода с функциями:
- Ввод многострочного кода
- Выполнение кода
- Вывод результатов
- Система подсказок
- Проверка правильности решения

#### DialogueManager
Система диалогов с NPC:
- Последовательные реплики
- Выбор вариантов ответа
- Интеграция с заданиями
- Анимация текста

#### SaveSystem
Сохранение прогресса игрока:
- Выполненные задания
- Текущая глава
- Позиция игрока
- JSON формат

## 🎯 Поддерживаемые Python конструкции

### Базовые операции
```python
# Переменные
x = 10
name = "Player"

# Вывод
print(x)
print("Hello, " + name)

# Математика
result = x + 5
```

### Условия
```python
if x > 5:
    print("Greater")
elif x == 5:
    print("Equal")
else:
    print("Less")
```

### Циклы
```python
# For loop
for i in range(5):
    print(i)

# While loop
count = 0
while count < 3:
    print(count)
    count = count + 1
```

### Списки
```python
numbers = [1, 2, 3, 4, 5]
print(numbers[0])
numbers.append(6)
```

## 📦 Установка и запуск

### Требования
- Unity 2021.x или выше
- Windows/Mac/Linux

### Запуск в редакторе
1. Откройте проект в Unity
2. Откройте сцену `Scenes/Menu.unity`
3. Нажмите Play

### Сборка игры
1. File → Build Settings
2. Добавьте сцены в правильном порядке:
   - Menu.unity
   - Chapter1.unity
   - Chapter2.unity
3. Выберите платформу
4. Build
